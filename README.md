# 숲속의 작은 용사 3D (Forest Hero 3D)

아이와 함께 즐길 수 있는 3D 픽셀 스타일의 벨트스크롤 액션 게임입니다. Three.js(WebGL)를 사용하여 도트 감성의 입체적인 세계를 구현했습니다.

## 🎮 게임 특징
- **3D Voxel 그래픽**: 입체적인 도트 스타일의 캐릭터와 숲 배경.
- **성장 시스템**: 몬스터를 처치하여 경험치를 얻고 레벨업하면 외형(뿔, 무기 크기)이 변하고 강력해집니다.
- **다양한 몬스터**: 슬라임, 늑대, 해골 등 각기 다른 패턴의 몬스터 등장.
- **아이템 시스템**: 몬스터 처치 시 낮은 확률로 포션(체력 회복)과 코인 드랍.
- **보스전 & 상점**: 스테이지 끝에 등장하는 거대 보스를 물리치고 상점에서 장비를 강화하세요.
- **월드 시스템**: 대지 밖으로 떨어지면 게임 오버되는 긴장감 넘치는 월드 구조.
- **감정 표현**: 몬스터가 공격할 때와 맞았을 때의 표정이 실시간으로 변합니다.

## 🕹️ 조작 방법
- **화살표 키 (←, ↑, ↓, →)**: 용사 이동
- **Space Bar**: 점프 (월드 밖으로 떨어지지 않게 조심하세요!)
- **Z 키**: 공격 (꾹 누르면 기를 모아 강력한 **차지 공격** 발동)
- **X 키**: 방패 방어 (적의 공격을 무효화)

## 🛠️ 기술 스택
- **Engine**: [Three.js](https://threejs.org/) (WebGL)
- **Audio**: Web Audio API
- **Language**: HTML5, JavaScript, CSS3

## 🚀 실행 방법
1. 이 프로젝트의 모든 파일(`index.html`, `js/game.js`, `css/style.css`)을 다운로드합니다.
2. 폴더 구조에 맞게 파일을 배치합니다.
3. `index.html` 파일을 웹 브라우저로 실행합니다.
4. 화면 중앙의 **[모험 시작]** 버튼을 클릭하면 사운드와 키보드 입력이 활성화됩니다.
